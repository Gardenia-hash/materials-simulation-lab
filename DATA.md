# Data Availability

This repository contains small synthetic demo datasets for code demonstration and portfolio purposes.

The full research dataset related to the current-driven Néel skyrmion transport study is hosted on Zenodo.

## Zenodo Dataset

Dataset title:

```text
Micromagnetic simulation dataset for current-driven Néel skyrmion transport in a granular ferromagnetic track
```

Zenodo DOI:

```text
https://doi.org/10.5281/zenodo.20179737
```

## Relationship Between This Repository and the Zenodo Dataset

The data files in this GitHub repository are lightweight synthetic examples.

They are used to demonstrate:

- Data loading
- Basic analysis workflows
- Plot generation
- Documentation structure
- Reproducible Python-based analysis

The complete research dataset is not stored directly in this GitHub repository.

Instead, the full dataset is archived on Zenodo and should be accessed and cited through the Zenodo DOI.

## Manuscript Status

The associated manuscript is currently under review.

The Zenodo dataset is cited in the Data Availability Statement of the manuscript.

This repository should currently be understood as a public analysis and documentation portfolio, not as the official final companion repository for a published paper.

After publication, this repository may be updated to include:

- Formal citation information for the published paper
- A `CITATION.cff` file
- Additional analysis scripts connected to the Zenodo dataset
- More detailed reproducibility instructions

## Data Usage Note

Users who want to use the full research dataset should cite the Zenodo record.

Please do not treat the synthetic demo data in this repository as the full research dataset.
