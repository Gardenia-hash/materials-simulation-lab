# Project Notes

This document records the planning and development notes for the Materials Simulation Lab repository.

## Purpose

The purpose of this repository is to build a public academic coding portfolio related to materials science, computational physics, and scientific data analysis.

This repository will focus on small, reproducible examples instead of private or unpublished research files.

## Main Research and Learning Directions

The repository may include projects related to:

- Micromagnetic simulation data analysis
- Skyrmion trajectory analysis
- Magnetization and hysteresis loop analysis
- Materials characterization data processing
- Semiconductor-related data analysis
- Python-based scientific visualization

## Possible Mini Projects

### 1. Hysteresis Loop Analysis

Analyze magnetization-field data and extract basic magnetic properties.

Possible outputs:

- Hysteresis loop plot
- Coercive field estimation
- Remanent magnetization estimation

### 2. Skyrmion Trajectory Analysis

Analyze simulated skyrmion motion from MuMax3 output data.

Possible outputs:

- Skyrmion position vs time
- Velocity calculation
- Trajectory visualization
- Hall angle estimation

### 3. Materials Characterization Data Demo

Analyze simple tabular data from materials characterization-style measurements.

Possible outputs:

- Line profile plots
- Distribution plots
- Basic statistics
- Curve fitting

## Repository Rules

Do not upload:

- Unpublished manuscript files
- Confidential research data
- Private proposal documents
- Supervisor or collaborator files
- Raw experimental data without permission
- Personal documents

Use only:

- Synthetic data
- Public demo data
- Self-written scripts
- Reproducible examples
- Learning notes

## Current Status

This repository is currently in the initial setup stage.

The first goal is to create a clean structure for future Python-based data analysis projects.
